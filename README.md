# AddisPay Demo

Welcome to the AddisPay Demo project! This is a modern, responsive web application built with React and TypeScript, designed to showcase the features and integration capabilities of the AddisPay payment platform.

## Features

-  **Responsive Design:** Works seamlessly on mobile, tablet, and desktop.
-  **Sidebar Navigation:** Easy access to documentation and integration guides.
-  **Dark/Light Mode:** Toggle between dark and light themes.
-  **Search Functionality:** Quickly find relevant information.
-  **Social & Support Links:** Connect via X, Slack, Telegram, and more.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/anaol216/addispay-demo.git
   cd addispay-demo
   ```

2. **Install dependencies:**
   ```sh
   npm install
   # or
   yarn install
   ```

3. **Start the development server:**
   ```sh
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser:**  
   Visit [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

## Project Structure

```
src/
  components/    # Reusable UI components (Sidebar, Header, Footer, etc.)
  pages/         # Application pages and routes
  assets/        # Images and static files
  styles/        # Global and component styles
```

## Customization

- **Logo:** Replace `/public/logo.png` and `/public/logo2.png` with your own branding.
- **Social Links:** Update social URLs in `Footer.tsx` and other relevant components.
- **Theme Colors:** Adjust Tailwind config or CSS variables for your brand palette.
